# Sitio web de Romina Ávila

Landing page estática y responsive, preparada para publicarse gratuitamente.

## Archivos

- `index.html`: estructura y contenido.
- `styles.css`: diseño responsive.
- `script.js`: menú móvil y año automático.
- `assets/romina-avila.webp`: fotografía optimizada.

## Ver el sitio en la computadora

Abrí `index.html` con el navegador.

## Publicar gratis en Netlify

1. Entrá a Netlify y creá una cuenta.
2. Elegí la opción para desplegar un sitio manualmente.
3. Arrastrá la carpeta completa `romina-avila-web`.
4. Netlify generará una dirección gratuita terminada en `.netlify.app`.
5. Esa dirección puede colocarse en la biografía de Instagram.

## Datos que conviene verificar antes de publicar

- Que el número `100838` corresponda efectivamente a una matrícula profesional y que la denominación sea la correcta.
- Protocolo y condiciones para acompañamiento de menores de edad.
- Que el teléfono y el usuario de Instagram sean definitivos.
- Políticas o requisitos profesionales aplicables a la jurisdicción.

Los colores principales se encuentran al inicio de `styles.css`, dentro de `:root`.
